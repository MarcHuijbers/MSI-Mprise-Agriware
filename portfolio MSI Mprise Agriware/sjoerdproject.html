Smart project (competentie 2)

Ik kijk erg tevreden terug naar het uitgevoerde project/onderzoek, ik heb hierbij ook daadwerkelijk het gevoel dat we iets hebben kunnen leveren waar ze bij Mprise mee verder kunnen. Het begin van het project was voor mij wel nog een groot vraagstuk. De vraag die ik hierbij had was hoe ik kon leveren aan het project als bedrijfskundige. Aan de start van het project werd me ook paar keer gevraagd: “wat leer je nou bij bedrijfskunde, en wat ga je daar nou later mee doen?”. Ik vond dit eerlijk gezegd een best moeilijke vraag, waar ik niet direct een antwoord op kon geven.

Nu ik zo terugkijk op dit project denk ik dat ik hier wel een antwoord op heb. Een bedrijfskundige is iemand die op elk type project vanuit uiteenlopende onderwerpen en vanuit verschillende aard een bijdrage kan leveren (competentie 4). Bij dit project was de aard van het project ook niet een onderwerp waar ik initieel veel verstand van had. Toch heb ik het gevoel dat ik als bedrijfskundige heb kunnen leveren aan het project. Door bijvoorbeeld het overzicht te kunnen blijven zien, en mijn groepsgenoten te blijven motiveren voor het project (competentie 6). Dit project was dan ook de eerste keer dat ik een project heb uitgevoerd met groepsgenoten vanuit verschillende disciplines. Dit vraagde merkbaar een andere manier van dingen doen dan dat ik gewend was.

Tegen de afronding van het project denk ik dat ik concreter mijn bedrijfskundige rol heb kunnen vervullen. Door het uitwerken van de risico's en de baten en lasten. Ook bij het formuleren van de vervolg stappen voor Mprise denk ik dat deze taken concreter zijn voor een bedrijfskundige.

Kijkend naar de cyclus hebben wij de fases onderzocht van het business probleem en het wiskundig probleem. Vervolgstappen voor Mprise zijn het implementatie probleem en het communicatieprobleem. Voornamelijk bij het communicatieprobleem kan een bedrijfskundige duidelijk zijn taak vervullen. Omdat het hier gaat over hoe het product gebracht wordt naar de eindgebruiker.

Al met al kijk ik dus tevreden terug naar dit onderzoek. Ik heb me tijdens project ontwikkeld voornamelijk op het softskills vlak. En heb een beter beeld gekregen wat ik als bedrijfskundige nu doe en kan. En waarbij ik mijn vaardigheden kan inzetten (competentie 4).
